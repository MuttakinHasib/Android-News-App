About the Files

Android News App is a mobile news system which run under Android platform that used for your own news application. With powerful and Responsive Admin Panel can manage news category, news item, app profile information, change admin username and password with generate password, etc. This application created by Android for client side and then PHP MySQL for Admin side. Run under Android platform which is the most popular operating system in the world. Using this application you can save your money and time in creating application for your own news application.

Getting Started

Before doing installation, you need to do the following things :
 
Have a web hosting to store admin panel files and Mysql database, to run this admin panel, web hosting that you use must have the following requirements :
  • PHP Version 5.4, 5.5 or 5.6
  • Support Mysqli
  • Support PDO
 
Because this documentation using cpanel, a web hosting with cpanel is recommended. If you don not have web hosting or domain, you can check on this sample site which provide hosting and domain service :
 
Free web hosting & domain services :
  • 000webhost
 
Premium web hosting & domain services (Highly Recommend) :

  • Domainesia (Indonesian Hosting)

Database Creation

1. Login to your cpanel hosting and open MySQL Databases
2. Create database and give database name like you want, as example : 'solodroid_news_db’
3. Create user for your database
4. Add user to your database and check ALL PRIVILEGES

Import Database .sql file

1. Open phpMyAdmin and select your database
2. Import file database android_news_app_db.sql  from download package inside 'admin_panel' folder
 
 
 

Upload PHP Scripts and Database Configuration

1. open File Manager via cpanel or ftp server, upload android_news_app.zip inside 'admin_panel' folder to your folder destination on your hosting, then extract it
2. to change configuration please see includes/config.php file
3. you must edit this data with your own data.
 
 
Default User Login Admin Panel
Username   : admin
Password    : admin
 

RTL (Right To Left) Mode

1. RTL (Right-to-left) means generally refers to support of languages that are written from right to left, example Language which using RTL mode e.g : Arabic Language, etc, if you want to Enable RTL mode in the admin panel side
2. see includes/config.php file and set value in the $ENABLE_RTL_MODE = 'true';
3. if you want to disable RTL mode, set to false value.

Check Database Connection

Before you check this make sure that your web hosting able to run .htaccess file 
 
1. check database connection, make sure your web response {"status":"ok","database":"connected"} 
2. url format to check database connection : http://<your_admin_panel_url>/api/check_connection
